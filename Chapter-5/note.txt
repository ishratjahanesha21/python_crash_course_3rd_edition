# Section 1 — What is an if Statement? (1:00)

An **if statement** allows Python to make decisions.

Think about real life.

If it is raining,
take an umbrella.

If your phone battery is below 20%,
charge it.

Programming works exactly the same way.

General syntax:

```python
if condition:
    # code to execute
```

Notice the colon.

Also notice the indentation.

Python uses indentation to know which code belongs inside the if statement.

Example:

```python
age = 20

if age >= 18:
    print("You can vote.")
```

Output

```
You can vote.
```

Since the condition is True,
Python executes the code.

---

# Section 2 — Conditional Tests (3:00)

Before Python can make a decision,
it first checks whether something is True or False.

These are called **conditional tests.**

Example:

```python
age = 18

print(age == 18)
```

Output

```
True
```

The double equal sign

```python
==
```

means

"Is it equal?"

It does NOT assign a value.

Remember

```python
=
```

means assignment.

```python
==
```

means comparison.

This is one of the most common beginner mistakes.

# Section 3 — Multiple Conditions (7:00)

Sometimes one condition isn't enough.

Python gives us

## and

Both conditions must be True.

Example

```python
age = 25
has_license = True

if age >= 18 and has_license:
    print("You can drive.")
```

Both are True.

So Python prints the message.

---

## or

Only one condition needs to be True.

Example

```python
day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("Weekend!")
```

Output

```
Weekend!
```

# Section 7 — if else (14:30)

Sometimes there are only two possibilities.

Example

```python
age = 16

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

Output

```
Minor
```

If the first condition is False,
Python automatically executes the else block.

---

# Section 8 — if elif else (16:30)

What if there are multiple possibilities?

Example

```python
score = 82

if score >= 90:
    print("Grade A")
elif score >= 80:
    print("Grade B")
elif score >= 70:
    print("Grade C")
else:
    print("Need Improvement")
```

Python checks from top to bottom.

As soon as one condition becomes True,

it stops checking the rest.

This is very important.

---

# Section 9 — Multiple Independent if Statements (19:00)

Sometimes you don't want Python to stop.

Example

```python
age = 25

if age >= 18:
    print("Adult")

if age >= 21:
    print("Can enter club")
```

Output

```
Adult
Can enter club
```

Since these are separate if statements,

both conditions are checked.

# Section 10 — Using if with Lists (21:00)

Suppose we have

```python
requested_toppings = [
    "mushrooms",
    "cheese",
    "pepperoni"
]
```

Loop through them.

```python
for topping in requested_toppings:

    if topping == "cheese":
        print("Adding extra cheese.")

    else:
        print(f"Adding {topping}.")
```

This combines loops and conditions.

You'll use this technique constantly in real projects.

---

# Section 11 — Empty Lists (24:00)

Sometimes a list has no items.

Instead of checking the length,

Python lets us do this.

```python
users = []

if users:
    print("Users found.")
else:
    print("No users available.")
```

Very clean.

Very Pythonic.

---

# Section 12 — Multiple Lists (25:30)

Example

```python
available = ["apple", "banana", "orange"]

requested = ["banana", "grape"]

for item in requested:

    if item in available:
        print(f"{item} is available.")

    else:
        print(f"{item} is unavailable.")
```

This is commonly used in

shopping carts,

inventory systems,

and databases.
