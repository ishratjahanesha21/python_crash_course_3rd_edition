## What is a Python List?

A Python list is a collection that can store multiple items in a single variable.

Think of it like a shopping basket.

Instead of carrying each item separately, you place everything inside one basket.

For example, if you have three favorite fruits, instead of writing:

```python
fruit1 = "Apple"
fruit2 = "Banana"
fruit3 = "Orange"
```

You can simply write:

```python
fruits = ["Apple", "Banana", "Orange"]
```

Now all three values are stored in one variable called **fruits**.

---

## Characteristics of Lists

Python lists have several important features.

### Ordered

Lists keep items in the order you add them.

```python
colors = ["Red", "Green", "Blue"]
```

The first item is Red, the second is Green, and the third is Blue.

Python remembers this order.

---

### Mutable

Lists are mutable.

That means you can change them after they're created.

For example:

```python
fruits = ["Apple", "Banana", "Orange"]

fruits[1] = "Mango"

print(fruits)
```

Output:

```python
['Apple', 'Mango', 'Orange']
```

The second item was changed from Banana to Mango.

---

### Allows Duplicate Values

Lists can contain duplicate items.

```python
numbers = [5, 5, 8, 8, 10]
```

Python allows this without any problem.

---

### Can Store Different Data Types

A list doesn't have to contain only one type of data.

For example:

```python
data = ["Alice", 22, True, 95.5]
```

This list contains:

* a string
* an integer
* a boolean
* and a float

---

## Creating Lists

Creating a list is simple.

Use square brackets.

```python
numbers = [1, 2, 3, 4, 5]
```

An empty list:

```python
items = []
```

Or using the list constructor:

```python
items = list()
```

---

## Accessing List Items

Each item has an index.

Indexes start from zero.

```python
fruits = ["Apple", "Banana", "Orange"]
```

Index positions are:

Apple → 0

Banana → 1

Orange → 2

Access the first item:

```python
print(fruits[0])
```

Output:

```
Apple
```

Second item:

```python
print(fruits[1])
```

Output:

```
Banana
```

---

## Negative Indexing

Python also supports negative indexes.

```python
print(fruits[-1])
```

Output:

```
Orange
```

Here:

* -1 means last item
* -2 means second last
* -3 means third last

This is extremely useful.

---

## List Slicing

Slicing allows us to get multiple items.

```python
numbers = [10,20,30,40,50]
```

```python
print(numbers[1:4])
```

Output:

```
[20,30,40]
```

Python starts at index 1 and stops before index 4.

---

## Changing List Items

Updating values is easy.

```python
colors = ["Red", "Green", "Blue"]

colors[2] = "Yellow"
```

Now the list becomes

```python
["Red","Green","Yellow"]
```

---

## Adding Items

### append()

Adds an item to the end.

```python
fruits.append("Mango")
```

Result:

```python
["Apple","Banana","Orange","Mango"]
```

---

### insert()

Insert at a specific position.

```python
fruits.insert(1,"Grapes")
```

Result:

```python
["Apple","Grapes","Banana","Orange"]
```

---

### extend()

Combine two lists.

```python
list1 = [1,2,3]
list2 = [4,5]

list1.extend(list2)
```

Result:

```python
[1,2,3,4,5]
```

---

## Removing Items

### remove()

Removes by value.

```python
fruits.remove("Banana")
```

---

### pop()

Removes by index.

```python
fruits.pop(1)
```

Without an index:

```python
fruits.pop()
```

It removes the last item.

---

### del

Delete using an index.

```python
del fruits[0]
```

Delete the entire list:

```python
del fruits
```

---

### clear()

Removes everything.

```python
fruits.clear()
```

The list becomes:

```python
[]
```

---

## Finding the Length

Use the `len()` function.

```python
fruits = ["Apple","Banana","Orange"]

print(len(fruits))
```

Output:

```
3
```

---

## Looping Through a List

Using a for loop:

```python
fruits = ["Apple","Banana","Orange"]

for fruit in fruits:
    print(fruit)
```

Output:

```
Apple
Banana
Orange
```

This is one of the most common uses of lists.

---

## Checking if an Item Exists

Use the `in` keyword.

```python
if "Apple" in fruits:
    print("Found!")
```

Output:

```
Found!
```

---

## Sorting Lists

Sort numbers:

```python
numbers = [5,2,8,1]

numbers.sort()
```

Output:

```
[1,2,5,8]
```

Reverse sorting:

```python
numbers.sort(reverse=True)
```

Output:

```
[8,5,2,1]
```

---

## Reversing a List

```python
numbers.reverse()
```

This changes the order of the list.

---

## Copying Lists

Instead of writing

```python
list2 = list1
```

Use

```python
list2 = list1.copy()
```

This creates an independent copy.

---

## Nested Lists

Lists can contain other lists.

```python
matrix = [
    [1,2,3],
    [4,5,6],
    [7,8,9]
]
```

Access an item:

```python
print(matrix[1][2])
```

Output:

```
6
```

## Practical Example

Let's create a simple student list.

```python
students = []

students.append("Alice")
students.append("Bob")
students.append("Charlie")

print(students)
```

Output:

```
['Alice', 'Bob', 'Charlie']
```

Remove a student:

```python
students.remove("Bob")
```

Now:

```
['Alice', 'Charlie']
```

This simple concept is used in real-world applications like:

* Student Management Systems
* E-commerce Shopping Carts
* Contact Lists
* Inventory Management
* Task Management Apps
* Social Media Applications

## Common List Methods

Here are some of the most useful list methods you'll use frequently:

| Method    | Purpose                              |
| --------- | ------------------------------------ |
| append()  | Add an item to the end               |
| insert()  | Add an item at a specific position   |
| extend()  | Add all items from another list      |
| remove()  | Remove an item by value              |
| pop()     | Remove an item by index              |
| clear()   | Remove all items                     |
| sort()    | Sort the list                        |
| reverse() | Reverse the order of items           |
| copy()    | Create a copy of the list            |
| count()   | Count how many times a value appears |
| index()   | Find the position of an item         |
